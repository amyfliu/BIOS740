## BIOS740 - HW03

This assignment focuses on implementing Graph Convolutional Networks (GCNs) for graph classification using the ENZYMES dataset from PyTorch Geometric. You will explore graph structures, preprocess the data, implement GCN models (both manually and using PyG), and evaluate their performance.

### Assignment Details

Please refer to the **GCN-by_hand-and-PyG-Assignment.ipynb** notebook for the completed implementation and detailed results. This notebook contains:

- **Manual GCN Implementation**: Step-by-step construction of Graph Convolutional Networks from scratch
- **PyG-based GCN**: Implementation using PyTorch Geometric's built-in modules
- **Data Preprocessing**: Loading and preparing the ENZYMES dataset
- **Model Training & Evaluation**: Performance metrics and comparative analysis
- **Visualizations**: Graph structures and classification results

The notebook demonstrates both the theoretical foundations and practical implementation of GCNs for graph-level classification tasks.
